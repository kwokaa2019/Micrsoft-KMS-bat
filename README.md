步驟一：請先用Windows 內置的記事本打開windows 10｜11專用.bat ，


步驟二：請按Ctrl+H 在[尋找目標]輸入"192.168.1.1"，[取代為] 輸入 "KMS.myftp.biz"

步驟三：然後選擇「檔案」>「儲存檔案」，或選擇「另儲為」>檔案名打Kms，副檔名為.bat的

編碼記得必須要係ANSI

步驟四：請將滑鼠移至剛剛Save 的 .bat檔案上面，按右鍵選擇「以系統管理員身份執行」。(此批次檔會自動執行)
